## Floren Hasher

Floren Hasher проверяет хеш файлы только для файлов от https://github.com/Floren-Team

Как пользоваться?

1. В странице релизы размещены хеши для проверки

florenhasher.exe scanall - Проверить все

Если в результате написано RID Неверный значит версия неподходит

Если хеш неверный то вы скачали что то другое а не от https://github.com/Floren-Team
